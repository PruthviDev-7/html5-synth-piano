## ‚ú® Features
* **Zero Latency:** Sounds are generated instantly; no waiting for audio files to load.
* **Single File:** All code (HTML, CSS, JS) lives in one `index.html` file.
* **Polyphonic:** Play multiple keys at once.
* **Responsive Design:** Works on desktop (mouse/keyboard) and mobile (touch).
* **Visual Feedback:** Active states and smooth CSS transitions.
* **Audio Filtering:** Uses a Low Pass Filter for a warm, non-robotic sound.
